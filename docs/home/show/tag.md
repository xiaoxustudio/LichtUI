## tag Attributes

| 参数     | 说明     | 类型    | 可选值                                                | 默认值    |
| -------- | -------- | ------- | ----------------------------------------------------- | --------- |
| type     | 类型     | string  | default / primary / success / warning / info / danger | `default` |
| theme    | 主题     | string  | light / dark/ plain                                   | `light`   |
| closable | 可关闭的 | boolean |                                                       | `false`   |
