## Input Attributes

| 参数     | 说明 | 类型    | 可选值                                                | 默认值    |
| -------- | ---- | ------- | ----------------------------------------------------- | --------- |
| disabled | 禁用 | boolean |                                                       | `false`   |
| default  | 默认 | boolean |                                                       | `false`   |
| type     | 类型 | string  | default / primary / success / warning / info / danger | `default` |
