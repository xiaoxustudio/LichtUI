## Button Attributes

| 参数     | 说明     | 类型    | 可选值                                                | 默认值    |
| -------- | -------- | ------- | ----------------------------------------------------- | --------- |
| type     | 类型     | string  | default / primary / success / warning / info / danger | `default` |
| disabled | 禁用状态 | boolean | true / false                                          | `false`   |
| round    | 圆角     | boolean | true / false                                          | `false`   |
| plain    | 无装饰   | boolean | true / false                                          | `false`   |
