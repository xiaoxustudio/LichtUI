## Row Attributes

| 参数      | 说明           | 类型             | 可选值       | 默认值   |
| --------- | -------------- | ---------------- | ------------ | -------- |
| gutter    | 栅格间隔       | number           |              | `0`      |
| direction | 排列方向       | string           | row / column | `column` |
| truncated | 截断或截断行数 | boolean / number | row / column | `false`  |
